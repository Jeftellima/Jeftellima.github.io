Passo a Passo do Projeto:
Passo 1: Entrar no sistema

Ao abrir o sistema, você verá uma tela com dois painéis: o logotipo do sistema no lado esquerdo e, à direita, os campos para login e cadastro


    Username
    Password

Passo 2: Fazer login no sistema

A -No painel direito, você verá dois campos: Username e Password.
B -Digite seu nome de usuário no campo “Username” e sua senha no campo “Password”.
C -Obs: A senha aparecerá como asteriscos (*) para manter sua privacidade.
D -Clique no botão “Login” para tentar acessar o sistema.
E -Se o login for bem-sucedido, você verá uma mensagem de boas-vindas.
F -Caso o login não seja bem-sucedido, verifique suas informações e tente novamente.

Passo 3: Cadastrar-se no Sistema (Caso Ainda Não Tenha Realizado o registro)

A -Se você ainda não tem uma conta, clique no botão “Register” (Registrar) na parte inferior do painel.
B - tela mudará para um formulário de cadastro. Preencha os seguintes campos: Name: Seu nome completo. Email: Seu endereço de e-mail. Username: O nome de usuário que você deseja criar. Password: Uma senha de sua escolha.
C -Depois de preencher todos os campos, clique no botão “Register” (Registrar).
D -Se todas as informações estiverem corretas, você verá uma mensagem confirmando que sua conta foi criada com sucesso..

Passo 4: Retornar à Tela de Login

A -Após o registro, clique no botão “Back” (Voltar) para retornar à tela inicial de login.
B -A Tela mudará para um formulário de cadastro. Preencha os seguintes campos: Agora você pode usar o nome de usuário e a senha criados para acessar o sistema..

Dicas Importantes: sistema de loginbr>
A -Esqueceu a senha?: Neste sistema, não há recuperação de senha automática. Se necessário, consulte o administrador do sistema.
B -Preenchimento Completo: Certifique-se de que todos os campos estão preenchidos ao se registrar, para evitar mensagens de erro.
C -Saída do Sistema: Para sair, basta fechar a janela.

Desenvolvendo esse código, aprendi a manipular interfaces gráficas com o Tkinter e a gerenciar dados de usuários usando o SQLite. Essa experiência aprimorou minhas habilidades em autenticação e criação de formulários de cadastro.
